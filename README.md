# Assignment 3

## Install requirements:
- git
- Python 3.10 (May work on other versions)
- Docker

## How to install

### Repository
#### Clone source code
<code> git clone https://github.com/VilPu/Data_intensive_systems-Assignment_3.git </code>

### Python
#### Setup Python virtual environment
<code> python3 -m venv .venv </code>

#### Activate the virtual environment
<code> source .venv/bin/activate </code>

#### Install requirements.txt
<code> pip install -r requirements.txt </code>


### Docker
#### Start the postgres containers
<code> docker compose up -d </code>

## Run the App
After installation steps you should be able to run the application with this command:

<code> flask run  </code>

The application is running on:
[127.0.0.1:5000](http://127.0.0.1:5000)
